知到、智慧树！简单几步教你如何自己刷课，如何科学上网！

发表于 2019-10-27 分类于 科学上网
本文字数： 1.2k 阅读时长 ≈ 1 分钟

大学开学这么久了，大家肯定还有很多网课没有看完，相信大家也是很烦恼；所以我决定尽我自己的微薄之力，给大家分享一下如何靠自己刷完“知到、智慧树”的课程！

需要使用到的工具

电脑 + Chrome浏览器 + 校园网 + 足够的电源

Chrome浏览器即谷歌浏览器，点我下载

如何科学上网

何为科学上网？

科学上网是指通过某种科学的方法绕过上网限制访问，就是我们常说的翻墙

谷歌浏览器之所以强大是因为谷歌浏览器具有丰富的、免费的插件供用户使用，大大提升了用户的使用体验。但是美中不足的是，因为防火墙的限制，我们不能使用这么丰富的插件库。不过！不用担心，我在这里给大家介绍科学上网的方法 ！

安装好谷歌浏览器之后，需要一个谷歌访问助手这样的插件，访问下面的百度网盘链接，下载到自己的电脑（注意：放在一个显而易见的文件夹，并标注好这是个啥东西）

访问：链接：https://pan.baidu.com/s/1ZQiAkzd7UC16ofr0Jri7zA 
     提取码：dlss

这个文件是个zip压缩文件，看官需要解压一下，放在你事先创建好的文件夹，之后打开Chorme，点击右上角的设置图标 –> 更多工具 –> 扩展程序

打开之后应该是这个亚子的（第一次安装科学上网的童靴，可能没有如图这么多插件）：

然后点击右上角的开发者模式点击打开之后，在界面上方会出现如图所示的三个选项：

然后选择第一个加载已解压的扩展程序，然后选中你刚刚解压的那个文件夹点确定，然后确保出现以下红框图标就可以了

这时候谷歌访问助手就已经安装成功了，此时此刻，你可以随意浏览谷歌系的一些网页，比如：谷歌地球（看遍世界各个角落）、谷歌学术（这里有很多大牛教授等写的论文）、谷歌搜索（此时此刻你可以使用谷歌搜索来搜索你想要的东西了~）、维基百科（一个类似百度百科的网站）等等等等……不再一一介绍。

如何刷课！！！！

言归正传，接下来是如何刷课的教程，刷课需要借助一些人写的脚本，这里给大家推荐一款插件Tampermonkey，这个插件汇集了大量的脚本，用户只需要找到自己想要的脚本然后点击安装就可以了！

Tampermonkey插件在浏览器页面的左上角应用里面搜索下载，过程简单不再介绍。

安装好Tampermonkey插件之后，给大家介绍个“脚本大全网站”

https://greasyfork.org/zh-CN/scripts

前往这个网站，在搜索的地方输入智慧树
